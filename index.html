IVR Project

Este proyecto implementa un sistema IVR (Interactive Voice Response) utilizando Flask, SQLite y la API de Nexmo/Vonage.

Descripción del Proyecto

El sistema IVR permite manejar llamadas telefónicas de manera automatizada, proporcionando opciones de menú y verificaciones de código OTP.

Instalación

git clone https://github.com/tu-usuario/ivr-project.git
cd ivr-project
pip install -r requirements.txt

Uso

# Configura los archivos JSON en ivr_templates/templates.json y ivr_templates/config.json
# Luego, ejecuta la aplicación Flask:
python app.py

Flujo de Llamadas

El siguiente es un ejemplo de flujo de llamadas manejado por el sistema IVR:

  • Bienvenida
  • Solicitud de OTP
  • Verificación del OTP
  • Confirmación del OTP
